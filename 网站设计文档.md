> 版本：v3.1 | 日期：2026-04-30
> 更新：确认用户系统、多角色后台、小程序同步、腾讯云存储

---

## 一、项目概述

### 1.1 目标
打造一个**品牌设计服务电商平台**，全面学习 Apple 官网的设计美学和交互体验。客户能像在 Apple Store 买 iPhone 一样——浏览服务、查看详情、加入购物袋、在线支付或预约商务咨询。

### 1.2 核心体验
- **浏览**：3秒内理解"有思网是做什么的"
- **选购**：像逛 Apple Store 一样浏览服务，按需自由组合
- **信任**：案例、数据、客户证言建立专业感
- **下单**：在线支付（已明确需求）或 商务咨询（需要沟通）

### 1.3 目标用户
- 主要：成都地区幼儿园园长、行政负责人
- 特征：非技术背景、注重品牌口碑、决策需参考案例
- 设备：70%+ 移动端访问（微信内打开）

## 三、页面详细设计

### 3.1 首页（Landing Page）

参照 Apple 首页的模块化滚动叙事结构：

| 区域 | 内容 | Apple 对标 |
|------|------|-----------|
| Hero区 | 全屏大图 + 主标题 + 双CTA | Apple.com 首屏 hero，大标题 + "了解更多" / "购买" 双链接 |
| 信任条 | 3个数据指标 | Apple "Why Buy at Apple" 条带 |
| 品类入口 | 6个品类卡片网格 | Apple 产品线卡片（Mac/iPad/iPhone...） |
| 套餐推荐 | 三档方案对比 | Apple iPhone 对比页 |
| 案例精选 | 图片轮播 | Apple 产品 gallery |
| 流程说明 | 4步图标流程 | Apple "Getting Started" 区域 |
| 客户评价 | 证言卡片 | Apple "Significant Others" |
| 底部CTA | 咨询表单 | Apple Footer 上方 CTA 横幅 |

**Hero区设计**：
- 背景：高品质幼儿园品牌案例全屏图（暗色叠加，白色文字）
- 主标题：`font-size: 56px; font-weight: 700; letter-spacing: -0.003em;`
- 副标题：`font-size: 28px; font-weight: 400; color: #86868B;`
- 双CTA：左侧"浏览服务"文字链接（蓝色 + >） / 右侧"立即购买"填充按钮

**品类卡片设计**：
- 2列3行网格（桌面端），1列（移动端）
- 每张卡片：白色背景、圆角 20px、内部 padding 48px
- 图标：品类 SVG 图标，48×48，stroke 风格
- 标题：`font-size: 28px; font-weight: 600;`
- 描述：`font-size: 17px; color: #86868B;`
- 底部链接："探索 >" 蓝色文字链接
- Hover：卡片 `transform: scale(1.02)`，阴影加深，过渡 0.3s ease

### 3.6 结算页（Checkout）

**在线支付流程**：
```
填写信息 → 选择支付方式 → 确认订单 → 支付 → 成功页
```

页面布局：
- 左侧：订单明细（服务列表 + 总价）
- 右侧：表单（姓名、幼儿园、手机、微信号、备注）
- 底部：支付方式选择（微信支付 / 支付宝）+ "确认支付"按钮
- 支付成功页：订单号 + "我们将在24h内联系您确认需求"

**商务咨询流程**：
```
填写信息 → 提交 → 成功页（商务人员48h内联系）
```

页面布局：
- 简洁表单：姓名、幼儿园、手机/微信号、感兴趣的服务（多选）、补充说明

### 4.10 滚动行为

| 行为 | 实现 |
|------|------|
| 区域入场 | Intersection Observer，元素进入视口 20% 时触发 `visible` class |
| Stagger 效果 | 子元素依次延迟 100ms 入场 |
| 图片加载 | 渐进式加载：模糊占位 → 清晰图（参照 Apple 的 progressive image） |
| 平滑滚动 | `scroll-behavior: smooth`，锚点导航 |
| 导航吸顶 | `position: fixed` + 毛玻璃背景 |
| 底部操作栏 | 滚动超过 Hero 后显示，`position: fixed; bottom: 0;` |

### 5.3 数据库设计

```sql
-- 服务分类表
CREATE TABLE categories (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  description TEXT,
  icon TEXT,
  hero_image TEXT,                    -- 品类页 Hero 图
  sort_order INT DEFAULT 0,
  nav_items JSONB
);

-- 服务表
CREATE TABLE services (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  category_id UUID REFERENCES categories(id),
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  tagline TEXT,
  description TEXT,
  price DECIMAL(10,2),
  price_unit TEXT DEFAULT '起',
  hero_image TEXT,                    -- 服务详情页 Hero 图
  features JSONB,                     -- [{icon, title, desc}]
  includes JSONB,                     -- [{name, spec, icon}]
  specs JSONB,                        -- 参数 [{label, value}]
  delivery_days INT,
  warranty_days INT,
  related_service_ids UUID[],
  related_case_ids UUID[],
  is_active BOOLEAN DEFAULT true,
  sort_order INT DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 套餐表
CREATE TABLE packages (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  tagline TEXT,
  price DECIMAL(10,2),
  original_price DECIMAL(10,2),
  is_recommended BOOLEAN DEFAULT false,
  features JSONB,
  service_ids UUID[],
  delivery_days INT,
  warranty_days INT,
  is_active BOOLEAN DEFAULT true,
  sort_order INT DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 案例表
CREATE TABLE cases (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  category TEXT,
  service_ids UUID[],
  description TEXT,
  cover_image TEXT,
  gallery JSONB,
  before_images JSONB,
  after_images JSONB,
  testimonial TEXT,
  testimonial_author TEXT,
  delivery_days INT,
  is_featured BOOLEAN DEFAULT false,
  sort_order INT DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 用户表（客户）
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  phone TEXT UNIQUE,                   -- 手机号登录
  wechat_openid TEXT,                  -- 微信登录
  name TEXT,
  kindergarten TEXT,
  avatar TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 管理员表
CREATE TABLE admins (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  name TEXT NOT NULL,
  role TEXT NOT NULL DEFAULT 'admin',  -- super_admin / admin
  categories UUID[],                   -- 负责的品类ID列表（super_admin 为空=全部）
  permissions JSONB,                   -- 细粒度权限
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);
-- super_admin：可管理所有内容 + 系统设置 + 管理员管理 + 价格设置
-- admin：只能管理分配给自己的品类下的服务、案例、订单

-- 订单表
CREATE TABLE orders (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_no TEXT UNIQUE NOT NULL,
  user_id UUID REFERENCES users(id),   -- 关联用户（可选，游客下单为空）
  name TEXT NOT NULL,
  kindergarten TEXT NOT NULL,
  phone TEXT,
  wechat TEXT,
  items JSONB NOT NULL,
  total_price DECIMAL(10,2),
  message TEXT,
  order_type TEXT DEFAULT 'payment',  -- payment(在线支付) / consult(商务咨询)
  payment_method TEXT,                 -- wechat / alipay
  payment_status TEXT DEFAULT 'pending', -- pending/paid/refunded
  status TEXT DEFAULT 'pending',
  assigned_admin_id UUID REFERENCES admins(id), -- 分配给哪个管理员处理
  source TEXT DEFAULT 'website',
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

-- FAQ表
CREATE TABLE faqs (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  question TEXT NOT NULL,
  answer TEXT NOT NULL,
  sort_order INT DEFAULT 0,
  is_active BOOLEAN DEFAULT true
);

-- 网站设置表
CREATE TABLE site_settings (
  key TEXT PRIMARY KEY,
  value JSONB NOT NULL
);
```
