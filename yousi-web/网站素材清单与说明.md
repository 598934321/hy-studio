# 有思网 - 网站素材清单与说明文档

> 更新日期：2026-04-30
> 技术栈：Next.js 16 + TypeScript + Tailwind CSS + Zustand
> 设计风格：Apple 官网（黑白极简、毛玻璃、滚动动画）

---

## 一、项目概览

有思网是面向幼儿园的品牌设计服务电商平台，对标 Apple 官网设计风格。

### 网站结构（35个路由）

| 板块 | 路由 | 说明 |
|------|------|------|
| 首页 | `/` | Hero + 服务分类卡片 + 流程 + 评价 + CTA |
| 品牌服务 | `/services`, `/services/[slug]` | 品牌策略、视觉设计、内容创作 |
| IP服务 | `/ip`, `/ip/[slug]` | IP形象设计、授权运营、衍生品开发 |
| 音乐服务 | `/music`, `/music/[slug]` | 商业配乐、音乐制作、音效设计 |
| 视频服务 | `/video`, `/video/[slug]` | 企业宣传片、产品广告、短视频 |
| 技术支持 | `/tech`, `/tech/[slug]` | 网站开发、小程序、系统定制 |
| 套餐 | `/packages` | 轻量版/标准版/尊享版 对比 |
| 案例 | `/cases`, `/cases/[id]` | 案例列表（可筛选）+ 案例详情 |
| 专刊 | `/magazine` | 品牌洞察、行业趋势、设计灵感 |
| 内容页 | `/about`, `/faq`, `/contact`, `/terms` | 关于我们、FAQ、联系、条款 |
| 用户系统 | `/login`, `/account/*`, `/bag` | 登录注册、账户中心、购物袋 |
| 结算 | `/checkout`, `/checkout/consult`, `/checkout/success` | 下单、咨询、成功 |
| 管理后台 | `/admin/*` | 仪表盘、订单、服务、案例、咨询、设置 |

## 三、素材规格要求

### 图片格式
- **首选：** WebP（体积小，质量高）
- **备选：** JPEG（照片类）、PNG（需要透明背景时）
- **矢量：** SVG（Logo、图标）

### 图片尺寸
| 用途 | 建议尺寸 | 最小尺寸 |
|------|---------|---------|
| 全屏 Hero 背景 | 2880 x 1620 | 1920 x 1080 |
| 全宽分隔图 | 2880 x 1080 | 1920 x 720 |
| 服务主视觉 | 1920 x 1080 | 1280 x 720 |
| 案例封面 | 1920 x 1080 | 1280 x 720 |
| 案例展示图 | 1920 x 1080 | 1280 x 720 |
| 缩略图 | 800 x 600 | 400 x 300 |
| 文章封面 | 800 x 500 | 400 x 250 |

### 文件大小
- Hero 大图：< 500KB（WebP）/ < 1MB（JPEG）
- 服务主视觉：< 300KB
- 案例图：< 300KB
- 缩略图：< 100KB

## 七、技术说明

### 当前占位符系统
所有图片位置使用 `IMG-XXX` 编号占位符显示，页面上会显示灰色背景 + 编号文字。替换为真实图片时，需要：

1. 将图片上传到 `/public/images/` 对应目录
2. 修改对应 data 文件中的 `heroImage`/`coverImage` 字段为实际路径
3. 修改页面组件中的 ImagePlaceholder 为 `<img>` 或 Next.js `<Image>` 组件

### 深色模式
网站支持自动深色模式（跟随系统），图片素材建议：
- Hero 大图：选择深色调或可叠加半透明遮罩的图片
- 避免使用纯白色背景的图片（深色模式下会显得突兀）
